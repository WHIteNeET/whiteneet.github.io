「Note」如何阅读一本书

Posted on 2018-10-28 | In Note
我们的目标是————


没有蛀牙

为获取咨询而读，以及为求得理解而读

阅读的层次
1. 基础阅读
2. 检视阅读
3. 分析阅读
4. 主题阅读

检视阅读
その一

1. 先看书名页，有序看序
2. 研究目录
3. 如有索引，检阅之
4. 出版者介绍
5. 挑主题篇章来看
6. 随机翻阅，但不要在某一处看太多
7. 后记

その二
头一次看难读的书时，遇到不懂的地方不要停留思索或查询，继续读，从头到尾先读一遍。


控制阅读的速度


一个阅读者要提出的四个基本问题
1. 整体来说，这本书再说什么？
分析主题，列点
2. 作者细部说了什么，怎么说的？
从想法，声明，论点等角度来考虑
3. （完成1.2步之后）这本书说的有道理吗？是全部有？还是部分有？
4. 这本书与我有什么关系？


要读出作者的言外之意
保持清醒 思考 感想

如何做笔记
1. 写在目录的结构笔记
什么样的书?
书在讨论什么
作者所用的架构
2. 概念笔记
作者对该主题的想法+自己的观点
3. 辨证笔记
针对一场讨论情景的笔记

关于习惯
对单一步骤的练习

养成习惯

分析阅读的第一个阶段
四个规则

  1. 最好在阅读之前就知道自己在读什么，同时在自己心中对该种书有一个分类的标准。
    理论/知识——知道是怎么一回事
    实用——如果我们想做什么事，我们需要什么知识
    学习去怀疑。
    辨证看待书籍类型
    拿对理论性书籍举例：
    经验类别
    思考方式
    论证方式
    哪儿不同？具体到哪些词目，命题？

    学习以适当的态度来阅读不同的书

  2. 使用尽量少的句子概括全书
  3. 列举重要章节，说明他们是如何按顺序组成一个整体架构
    一堆转与一间房子的比喻
  4. 找出作者要问的问题（作者的意图）
    要注意”意图谬误“
    另:四个规则具有整体性，他们共有一个目标。
    掌握结构大纲

分析阅读的第二个阶段
同样是四个规则：

  1. 找出共通的词义
    抓出重要关键字，每个字的不同意义的转换
  2. 从最重要的句子中抓出作者的重要主旨
  3. 找出作者的论述，重新架构这些论述的前因后果，以明白作者的主张
  4. 确定作者已经解决了哪些问题，还有哪些是未解决的。在未解决的问题中，确定哪些是作者认为自己无法解决的问题。
    明确目的：诠释作者的意图

    阅读的一部分本质就是被困惑，而且知道自己被困惑。

最后一个阶段
公正地评断一本书

遵守思维的礼节
主动的阅读必须能评论，提出批评，才算真正完成了这件事。
一个人如果不能自动自发地运用独立的判断力，他根本就不可能学习到任何东西。或许他可以受训练，却不能受教。

  1. 在说出”我同意“，”我不同意“，或”我暂缓评论“之前，你一定要能肯定地说：”我了解了“。
  2. 当你不同意作者的观点时，要理性地表达自己的意见，不要无理地辩驳或争论。
  3. 尊重知识与个人观点的不同，再做任何评断之前，都要找出理论基础。

赞成或反对作者
理想化辩论的三个条件
1) 注意自己的情绪，理性辩论。
2) 把自己的前提或假设摊出来。
3) 为化解盲点，应尽力尝试不偏不倚。
如何向作者表示反对
1) 你的知识不足 —— 缺少某些与他想要解决的问题相关的知识
2) 你知识有错误 —— 这样的错误可能来自缺乏知识，但也可能远不止于此
3) 你不合逻辑/你的推论无法令人信服 —— 分为两种。其一为缺乏连贯，前言不搭后语，其二为事件变化的前后不一致。
4) 你的分析不够完整 ——
