「Note」如何阅读一本书

Posted on 2018-10-28 | In Note
我们的目标是————


—
>为获得资讯而读，以及为求得理解而读

阅读的层次
1. 基础阅读
2. 检视阅读
3. 分析阅读
4. 主题阅读

检视阅读
その一
先看书名页，有序看序 研究目录
如有索引，检阅之 出版者介绍
挑主题篇章来看 随机翻阅，但不要在某一处看太多
后记 その二
头一次看难读的书时，遇到不懂的地方不要停留思索或查询，继续读，从头到尾先读一遍。


控制阅读的速度


一个阅读者要提出的四个基本问题
1. 整体来说，这本书再说什么？
分析主题，列点
2. 作者细部说了什么，怎么说的？
从想法，声明，论点等角度来考虑
3. （完成1.2步之后）这本书说的有道理吗？是全部有？还是部分有？
4. 这本书与我有什么关系？


要读出作者的言外之意 保持清醒
思考 感想

如何做笔记
1. 写在目录的结构笔记
1) 什么样的书
2) 书在讨论什么
3) 作者所用的架构
2. 概念笔记
作者对该主题的想法+自己的观点
3. 辨证笔记
针对一场讨论情景的笔记

关于习惯
对单一步骤的练习
